# az-104-labs

Hands-on labs and notes following the AZ-104 Microsoft Azure Administrator learning path.  
This repository includes step-by-step exercises, screenshots, scripts, and detailed documentation focused on identity, Role-Based Access Control (RBAC), governance, networking, and core Azure services.  
These labs are designed to reinforce practical skills and understanding of Azure administration concepts, making it easier to prepare for real-world scenarios and the AZ-104 certification.

Laboratorios prácticos y notas siguiendo la ruta de aprendizaje AZ-104 de Microsoft Azure Administrator.  
Este repositorio incluye ejercicios paso a paso, capturas de pantalla, scripts y documentación detallada, centrados en identidad, control de acceso basado en roles (RBAC), gobernanza, redes y servicios principales de Azure.  
Estos laboratorios están diseñados para reforzar habilidades prácticas y comprensión de los conceptos de administración de Azure, facilitando la preparación para escenarios reales y para la certificación AZ-104.
